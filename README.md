# Rainy Days

An e-commerce storefront for outdoor clothing, built as a static website.

## Features

- Product browsing across multiple product pages
- Shopping cart with currency display
- Responsive navigation with hamburger menu for mobile
- About and Contact pages
- Checkout flow with a success confirmation page

## Pages

- `headerpage.html` — Homepage
- `productsite1.html` – `productsite7.html` — Individual product pages
- `about.html` — About page
- `Contact.html` — Contact page
- `checkout-page.html` — Checkout page
- `checkout-Succes.html` — Order confirmation page

## Getting Started

No build step is required. Open the project directly in a browser:

1. Clone the repository:

```bash
git clone https://github.com/johnruud3/Rayny-Days-final-main.git
```

2. Open `index.html` in your browser (it redirects automatically to the home page), or use a local development server such as the [VS Code Live Server extension]
